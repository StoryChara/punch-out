# Punch-Out !!

### Controles del Juego
-  ENTER para entrar al juego.
- ← para esquivar hacia la izquierda.
- → para esquivar hacia la derecha.
- ↑ para golpear al contrincante.
- ↓ para bloquear un golpe.
- R para reiniciar el juego.
